# vscode  shortcut
사용하는 단축어 모음

아래에 줄 복사 - ⇧ + ⌥ + DownArrow
위에 줄 복사 - ⇧ + ⌥ + UpArrow
자동 줄 바꿈 - ⌥ + Z

언어 모드 변경 - ⌘ + K then M
모든 명령 표시 - ⇧ + ⌘ + P

### 다중 커서 - multiple cursor select
#### 선택 영역
- 다음 항목 추가 - ⌘ + D
- 모든 항목 추가 - ⇧ + ⌘ + L
- 줄 끝에 커서 추가 - ⇧ + ⌥ + I

### 편집기 분할
- 오른쪽 그룹으로 이동 - ⌘ + \
> 기존에 편집기 분할에서 변경함

### 맥의 키보드 아이콘
⌘ (Cmd) : 커멘드(Command) 키
⌥ (Opt) : 옵션(option)키
⌃ (Ctrl) : 컨트롤키
↩ (return) : 리턴키
⇧ (Shift) : 시프트키
⌫ (Delete) : 딜리트키
⎋ (esc) : 이스케이프키(escape)키
⏏ (eject) : 추출키
